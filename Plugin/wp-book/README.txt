This pluging contains custom post type, custom taxonomies,meta-date box,widget,dashboard widget, custom admin settings page, short-codes.

All code is written in Object Oriented Programming with modular approach.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-book.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('wp_book_hook'); ?>` in your templates
